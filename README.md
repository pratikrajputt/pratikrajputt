## 🚀 Featured Projects

### 🤖 [PolicyMind — RAG Assistant](https://github.com/pratikrajputt/policy-rag-assistant)
> Grounded RAG assistant for company policy documents with hallucination control and evaluation framework.

- Built end-to-end RAG pipeline: document chunking → embeddings → ChromaDB → LLM answer generation
- Engineered prompts to minimize hallucinations — model responds safely when info is missing
- Implemented manual evaluation framework across fully/partially/non-answerable questions
- **Stack:** Python · LangChain · ChromaDB · Ollama (Llama3) · nomic-embed-text
